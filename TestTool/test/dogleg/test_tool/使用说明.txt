===============================================
电机控制工具 - 打包使用说明
===============================================

【方法一：使用批处理文件（最简单）】
1. 双击运行 build_exe_simple.bat
2. 等待打包完成
3. 在 dist 目录中找到生成的 exe 文件

【方法二：使用 Python 脚本】
1. 打开命令行（CMD 或 PowerShell）
2. 进入当前目录
3. 运行：python build_exe.py
4. 选择打包模式（1=单文件，2=文件夹）
5. 等待打包完成

【打包前准备】
1. 确保已安装 Python 3.7 或更高版本
2. 运行：pip install -r requirements.txt
   安装所有依赖包

【打包结果】
- 单文件模式：dist/电机控制工具.exe
- 文件夹模式：dist/电机控制工具/电机控制工具.exe

【在其他电脑上运行】
1. 将 exe 文件（或整个文件夹）复制到目标电脑
2. 直接双击运行即可
3. 无需安装 Python 或其他软件

【注意事项】
1. 如果程序需要连接 PCAN 设备，目标电脑需要安装 PCAN 驱动
2. 首次运行可能需要几秒钟启动时间
3. 如果被杀毒软件拦截，请添加信任

【常见问题】
Q: 打包失败怎么办？
A: 检查是否安装了所有依赖：pip install -r requirements.txt

Q: exe 文件太大？
A: 这是正常的，因为包含了 Python 解释器和所有库

Q: 在其他电脑无法运行？
A: 确保目标系统是 Windows 10 或更高版本
