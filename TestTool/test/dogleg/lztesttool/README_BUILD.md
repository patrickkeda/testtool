# 电机控制工具 - 打包说明

## 环境要求

- Python 3.7 或更高版本
- Windows 10/11
- 已安装所有依赖包

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行打包脚本

```bash
python build_exe.py
```

### 3. 获取打包结果

打包完成后，exe 文件位于 `dist` 目录中：
- **单文件模式**：`dist/电机控制工具.exe`（单个文件，约 50-100MB）
- **文件夹模式**：`dist/电机控制工具/电机控制工具.exe`（文件夹形式，启动更快）

## 打包选项说明

### 单文件模式（推荐）
- 优点：单个文件，便于分发和携带
- 缺点：首次启动较慢（需要解压临时文件）
- 适用场景：需要分发给其他用户

### 文件夹模式
- 优点：启动速度快，文件结构清晰
- 缺点：文件较多，需要整个文件夹一起分发
- 适用场景：自己使用或需要频繁运行

## 在其他电脑上运行

### 系统要求
- Windows 10 或更高版本
- 无需安装 Python 或其他依赖

### 使用方法
1. 将 `dist` 目录中的 exe 文件（或整个文件夹）复制到目标电脑
2. 直接双击运行 exe 文件
3. 首次运行可能需要几秒钟的启动时间

## 注意事项

### PCANBasic DLL
如果程序需要与 PCAN 设备通信，目标电脑需要：
1. 安装 PCAN 驱动程序（从 Peak System 官网下载）
2. 或者将 PCANBasic.dll 放在 exe 同目录下

### 常见问题

**Q: 打包后的 exe 无法运行？**
- 检查是否被杀毒软件拦截
- 确认目标系统是 Windows 10 或更高版本
- 查看是否有错误提示信息

**Q: 程序运行缺少 DLL？**
- 确保安装了 PCAN 驱动程序
- 尝试使用文件夹模式打包（包含更多依赖）

**Q: 打包后的文件太大？**
- 这是正常的，因为包含了 Python 解释器和所有依赖
- 单文件模式通常 50-100MB
- 文件夹模式可能更大，但启动更快

**Q: 如何添加图标？**
- 准备一个 .ico 格式的图标文件
- 修改 `build_exe.py` 中的 `--icon=图标文件.ico`

## 高级选项

如果需要自定义打包选项，可以：
1. 修改 `build_exe.py` 中的 PyInstaller 参数
2. 或使用 `pyinstaller` 命令直接打包：

```bash
pyinstaller --onefile --windowed tool.py
```

更多选项请参考 [PyInstaller 文档](https://pyinstaller.org/)
