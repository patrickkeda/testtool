# 快速开始 - VITA 工程模式测试

## 🚀 一分钟上手

### 方式1: 使用自动工程模式（推荐）

使用 `Vita01-MT-1.0-EngineerMode.yaml`：

```yaml
steps:
  - ScanSN              # 扫描序列号
  - ConnectEngineer     # 连接并自动进入工程模式
```

**优点**: 简单、稳定、推荐生产环境使用

---

### 方式2: 使用enfac命令

使用 `Vita01-MT-1.0-EnfacCommand.yaml`：

```yaml
steps:
  - ScanSN              # 扫描序列号
  - ConnectEngineer     # 连接工程服务
  - EnterEngineer       # 执行 enfac=1,1% 命令
```

**优点**: 灵活、可自定义、适合调试开发

---

## 📝 关键步骤说明

### Step 1: 扫描序列号
```yaml
- id: step_1
  name: ScanSN
  type: scan.sn
```

### Step 2: 进入工程模式

**方式A - 自动模式（推荐）**:
```yaml
- id: step_2
  name: EnterEngineerMode
  type: connect_engineer
  params:
    host: '192.168.125.2'
    port: 3579
    timeout_ms: 10000
```

**方式B - 显式命令**:
```yaml
- id: step_2
  name: ConnectEngineer
  type: connect_engineer
  params:
    host: '192.168.125.2'
    port: 3579

- id: step_3
  name: ExecuteEnfac
  type: engineer.command
  params:
    command: 'enfac=1,1%'
    timeout_ms: 5000
```

```yaml
    command: 'camera=3,rgb%'

  params:
    command: 'lidar=3,360,1000%'

- type: engineer.command
  params:
    command: 'enfac=1,0%'
```

## 🔧 配置IP地址

### 在YAML中配置（推荐）
```yaml
params:
  host: '192.168.125.2'
  port: 3579
```

### 在配置文件中配置
编辑 `Config/config.yaml`：
```yaml
ports:
  1:
    uut:
      tcp:
        host: "192.168.125.2"
        port: 3579
```

---

## ⚠️ 注意事项

1. **命令格式**: 工程服务命令必须以 `%` 结尾，如 `enfac=1,1%`
2. **连接顺序**: 必须先 `connect_engineer`，再执行 `engineer.command`
3. **超时设置**: 根据命令类型设置合理的超时时间
4. **测试完成**: 建议最后执行 `disconnect_engineer` 断开连接
